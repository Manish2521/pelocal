# To-Do List App (Flask)

A simple To-Do List web application built using **Flask** and **SQLite**.  
The application supports task management using REST-style routes and server-rendered templates.  
No ORM or generic viewsets are used, as per assignment requirements.

---

## Features

- Create, view, update, and delete tasks
- Server-rendered HTML templates
- SQLite database with raw SQL queries
- Logging with user IP address and timestamps
- Basic error handling
- Pytest-based API testing

---

## Tech Stack

- Python 3
- Flask
- SQLite
- Pytest
